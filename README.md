# Selvavel Website Rebrand

This project is a React-based website rebranded to align with `selvavel.com`, including:

- Updated brand messaging and tone around Data Engineering and AI Consulting
- Revised page hierarchy: Home, About, Services, Resources, Contact
- SEO-focused metadata (title, description, keywords, Open Graph, Twitter cards)
- Refreshed responsive design system (color palette, typography, section layout, CTA patterns)

## Development

Install dependencies and run locally:

```bash
npm install
npm start
```

Build for production:

```bash
npm run build
```
